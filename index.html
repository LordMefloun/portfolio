O mně

Jsem 17ti letý kluk, který se od malička zabývá počítači. První jazyk , který jsem kdy viděl byl Visual Basic .NET. Druhý „jazyk“ se kterým jsem se seznámil byl plugin Skript v Minecraftu.

Chtěl jsem však umět něco více. Naučil jsem se tedy základy HTML a CSS. Poté jsem se začal učit můj první programovací jazyk C# .NET.

Programovat konzolové a desktopové aplikace mě přestalo bavit. Tak jsem se učil jazyk PHP

Jako třetí programovací jazyk jsem se učil Javu, kvůli užití ve Spigot Minecraft pluginech

Programování Minecraft pluginů mě však také omrzelo, tak jsem se ještě více zaměřil na C# .NET, ve kterém jsem se naučil používat Socket Programování TCP/IP, WebSockety a také Multi Threading
